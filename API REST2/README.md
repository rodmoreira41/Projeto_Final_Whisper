# Desenvolvimento Web II

Neste repositório está o projeto desenvolvido para a disciplina 'Desenvolvimento Web II', uma disciplina do segundo ano do curso de informática da Universidade da Maia. Desenvolvido pelo Grupo _01_: [@AnaAzevedo](https://github.com/AnaAzevedo2), [@MarianaLopes](https://github.com/marlope02), [@MarioRodrigues](https://github.com/MarioRodrigues2304), [@Rodrigo Moreira](https://github.com/rodmoreira41).

## Tema 

Este trabalho tem como âmbito o desenho e criação de um serviço web que relaciona recursos. Tais recursos como 'Movies', 'Genres', 'Actor' e 'Director'. O recuso 'Movies' será o recurso principal, já que terá um genero, ator e diretor associado.
